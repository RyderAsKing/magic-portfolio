# Server Manager

Are you sick of having to log into hundreds of different websites just to access your server? Well, we got you! Server Manager is an open-source project that allows you to manage all your servers from one unified dashboard, regardless of their provider. Feel free to set up a local instance - it's incredibly helpful for managing multiple servers.

## Supported Platforms

Virtualizor ✅
Pterodactyl ✅

## Technical Implementation

### Stack & Technologies

- Laravel 8.0 PHP Framework
- React for dynamic frontend
- MariaDB for database
- Redis for caching
- Nginx web server
- WebSocket for real-time updates

### Key Features

- Unified dashboard for multiple server providers
- Real-time server status monitoring
- Secure authentication system
- API integration with multiple platforms
- Automated deployment options
- Comprehensive logging system

## Installation Guide

### System Requirements

- Ubuntu OS (recommended)
- PHP 8.0+
- Node.js & NPM
- MariaDB
- Redis
- Nginx

### Core Dependencies

- PHP Extensions: CLI, GD, MySQL, PDO, MBString, TokenizerBCMath, XML, FPM, CURL, ZIP
- Composer for PHP package management
- Node.js packages: Express, WS, Axios
- SSL Certificate (recommended)

### Security Features

- HTTPS support
- Environment-based configuration
- Database security best practices
- File permission management
- Regular security updates

Server Manager has significantly improved server management workflows:
