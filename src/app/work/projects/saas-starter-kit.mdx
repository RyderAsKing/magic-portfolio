# NertiaKit

A modern SaaS starter kit that combines Laravel's robust backend with Inertia.js and React for a seamless single-page application experience. Built with developer productivity and modern web standards in mind.

## Core Features

### Authentication & Access Control

- Built-in authentication using Laravel Breeze
- Role-Based Access Control (RBAC) with Spatie Permissions
- Protected routes and middleware setup
- User role management system

### Modern Frontend Stack

- Inertia.js + React integration
- ShadCN UI Components for modern styling
- TailwindCSS for responsive design
- TypeScript support for better type safety
- Toast notifications system

### Developer Experience

- Pre-configured layouts and dashboards
- Type-safe data sharing between backend and frontend
- Hot module replacement in development
- Comprehensive documentation

## Technical Stack

### Frontend

- React with TypeScript
- Inertia.js for seamless page transitions
- TailwindCSS for styling
- ShadCN UI component library
- Toast notification system

### Backend

- Laravel 11
- Spatie Permissions for RBAC
- Laravel Breeze for authentication
- Type-safe API endpoints

## Project Benefits

- **Quick Start**: Launch your SaaS product faster with pre-built components
- **Type Safety**: Catch errors early with full TypeScript support
- **Modern UI**: Professional look with ShadCN and TailwindCSS
- **Scalable**: Well-organized structure for growing applications
- **Secure**: Built-in authentication and authorization
